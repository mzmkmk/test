Linux指令
ls 列出目录下的文件
-l 以每行一个的格式列出文件
-a 列出包含隐藏文件的所有文件
-s 以文件大小顺序列出文件
-t 以修改日期顺序列出文件
-r 以倒序列出文件、

cd ~ 跳到自己的home目录
cd ../.. 跳到目前目录的上上两层

rm 删除文件、目录
-i 删除前提示（YES or NO）
-r 删除一个目录（含所有文件）
-f 强制执行删除操作

git指令
git status 查看仓库的状态
git ls-files 查看已经提交的文件
git add 添加到暂存区 （git add *.txt	git add .）
git commit -m ""只提交暂存区的内容，不会提交工作区的内容
git log 查看仓库提交历史记录（git log --oneline）
git reset --soft
git reset --hard
git reset --mixed
git rm <file> 把文件从工作区和暂存区同时删除
git rm --cached <file> 把文件从暂存区删除，但保留在当前工作区中
git rm -r * 删除该目录下的所有文件和子目录
删除后不要忘记提交

克隆仓库 git clone repo-address
推送更新内容 git push <remote> <branch>
拉取更新内容 git pull <remote>

添加远程仓库 git remote add <远程仓库别名> <远程仓库地址> git push -u <远程仓库名> <本地分支名>:<远程分支名>
查看远程仓库 git remote -v
拉取远程仓库内容 git pull <远程仓库名><远程分支名>:<本地分支名>

./ 用户当前所在目录

查看分支列表 git branch
创建分支 git branch <branch_name>
切换分支 git switch <branch_name>
合并分支 git merge <branch_name>
删除分支 git branch -D <branch_name>

中止合并 git merge --abort
